# docs-as-models-as-code

## Core idea
Coined by DOF team lead J, a philosophy that says that just like docs can be treated as code, docs can be treated as models, existing and version controlled and released alongside the rest of the model. The DOF initiative considers this practice essential to actualizing open source hardware.
